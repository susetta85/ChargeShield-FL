# ChargeShield-FL

ChargeShield-FL is an open-source framework for privacy auditing in Federated Learning-enabled Operational Technology (OT) environments.

## Objectives

- ML Plane abstraction
- Privacy auditing
- Federated Learning experimentation
- OT/ICS integration
- Smart Grid and EV charging use cases

## Project Status

Early development (Sprint 1).

## Roadmap

- Sprint 1: Framework foundations
- Sprint 2: ChargingNode architecture
- Sprint 3: Containerlab integration
- Sprint 4: Privacy auditing experiments
